# netflix movies and TV shows Data Analysis using SQL
![Netflix logo](https://github.com/akhiladubbaka/netflix_SQL_project/blob/main/netflix.jpg)


## 📌 Project Overview

This project focuses on analyzing the Netflix dataset using SQL. The goal is to explore the dataset, answer business-related questions, and gain insights into Netflix content through data querying and analysis.

The project covers data cleaning, data exploration, and advanced SQL queries involving aggregations, window functions, CTEs, string functions, and conditional logic.

---

## 🛠️ Tools & Technologies

* PostgreSQL
* SQL
* GitHub

---

## 📂 Dataset Information

The dataset contains information about Netflix Movies and TV Shows, including:

* Show ID
* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre
* Description

---

## 🎯 Project Objectives

The project aims to answer questions such as:

* Total number of Movies and TV Shows
* Most common content ratings
* Longest movie available on Netflix
* Content distribution by country
* Top directors and actors
* Year-wise content additions
* Genre analysis
* Content categorization based on descriptions
* Advanced business-oriented insights

## 📈 Sample Business Problems Solved

1. Count the number of Movies and TV Shows.
2. Find the most common rating for Movies and TV Shows.
3. Identify the longest movie.
4. Find content added in the last 5 years.
5. List all Movies directed by a specific director.
6. Find the top actors appearing in the most content.
7. Determine which countries produce the most Netflix content.
8. Categorize content as Good or Bad based on keywords in descriptions.
9. Analyze yearly content growth on Netflix.
10. Explore genre-wise content distribution.
